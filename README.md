# BlackSweet  Ecommerce de Roupa Masculina 

 Bem-vindo a BlackSweet, uma plataforma moderna e responsiva de e-commerce especializada em moda masculina. Este projeto foi desenvolvido para oferecer uma experiência de compra intuitiva, rápida e segura, com foco em estilo, praticidade e performance.

# funcionalidade 

Catálogo completo de roupas masculinas:
camisetas, calças, jaquetas, acessórios e mais
Filtros avançados por categoria, tamanho, cor,      preço e marca
Sistema de carrinho de compras com atualização dinâmica
Checkout seguro com integração a gateways de pagamento
Área do cliente com histórico de pedidos e gerenciamento de perfil
Painel administrativo para controle de estoque, pedidos e produtos
Layout responsivo para desktop e mobile

# Tecnologia Utilizadas

Front-end:
Html5,CSS

# Pré requesitos 

Uso apenas de internet e qualquer aparelho tecnologico da atualidade.
